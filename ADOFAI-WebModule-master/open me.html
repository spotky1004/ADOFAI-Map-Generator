If you are getting an error such as below one, you should use a host instead.

Error: Access to script at '...' from origin 'null' has been blocked by CORS policy: Cross origin requests are only supported for protocol schemes: http, data, chrome, chrome-extension, chrome-untrusted, https.

How to create a host by yourself using npm:
1. If you do not have node.js, install node.js.
2. Just in case, update your npm, npx, nodejs and everything related to it.
3. If you do not have http-server module, install it by doing:
$ npm i -g http-server
4. If you do have http-server module, type the command:
$ npx http-server -c-1
5. Open your web browser and go to localhost:8080. Done!
